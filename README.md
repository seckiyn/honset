# honset

# Token Types

- STRING_LITERAL

"It's a string."

- LIST

[This, is, a, list]

# Execution

## PRINT (STRING_LITERAL)

Prints the string literal into the screen

## MEM (STRING_LITERAL, LIST)

Will print the string literal and will ask for the things in the list.

## WORK (STRING_LITERAL)

Will print the string literal.
